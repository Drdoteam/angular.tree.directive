# AngularJS Tree Directive

'Drdoteam/angular.tree.directive' is forked from 'https://github.com/eu81273/angular.treeview'.  
'eu81273/angular.treeview' is stopped Oct, 2013.

# Features

 - Collapsibility
 - Selectability
 - Tree(\<ul\>), Node(\<li\>) Custom Class
